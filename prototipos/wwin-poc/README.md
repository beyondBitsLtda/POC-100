# Protótipo POC Wwin Safety

Abra `prototipos/wwin-poc/index.html` no navegador.

Não requer servidor.

## Cascata Unidade → Setor → Subsetor

Inclua o script e use os IDs esperados no formulário de Cadastro Básico:

```html
<select id="unidadeSelect" class="form-control">
  <option value="">Selecione</option>
</select>
<select id="setorSelect" class="form-control">
  <option value="">Selecione</option>
</select>
<select id="subsetorSelect" class="form-control">
  <option value="">Selecione</option>
</select>

<script src="assets/js/cascata-unidade-setor-subsetor.js"></script>
```
